And welcome to this totally amazing ExtJS demo. Some things to note:

  • The search box will search on multiple columns in the table at once. This means if you type in "kylir horton" you will find something. Or "doe 343-3288". Try it out!
  • Double click on a customer row to edit. Changes are also searchable.
  • This is written entirely with ExtJS 4.2 using their built-in MVC framework. It was fun building this!
  • I tested only with Chrome. I'm pretty confident it'll work in other browsers too...
