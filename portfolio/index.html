Web App User Interfaces

Here's a quick portfolio showcasing some of the user interfaces I've designed and implemented. You can probably tell they are all from the same product - Cityworks. I'll add other projects I've worked on in the future when I get a chance. These UIs were implemented using ASP.NET and C# on the back end. In some instances REST end point services that I designed are used to transfer data between the browser/phone and the back-end business logic. Browser frameworks used include jQuery, ExtJS, Bootstrap, Dojo, RequireJS, and ArcGIS.
